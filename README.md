nodeps

==============================
A general purpose Photoshop file parser to HTMl file.

## Installation

```sh
$ npm install nodeps -g
```

## Execution

Go to the psd file directory and enter the following command at the command line.

```sh
// PSD parser to HTMl and open in browser
$ nodeps yourfile.psd

```
