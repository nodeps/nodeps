nodeps
==============================
A general purpose Photoshop file parser to HTMl file.

## Installation

clone the repository and install the dependencies.

```sh
$ git clone git@github.com:nodeps/nodeps.git
$ cd nodeps
$ npm install 
```

## Execution

Replace node.psd with your file.

```sh
// PSD parser to HTMl
$ npm start
// view the html file
$ npm test

```
