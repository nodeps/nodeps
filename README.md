nodeps
==============================
A general purpose Photoshop file parser to HTMl file.

## Installation

Install it as a command line tool via `npm -g`.

```sh
git@github.com:nodeps/nodeps.git
cd nodeps
npm install 
```

## Execution

Copy a psd file to the root directory
Example: h5.psd

```sh
// PSD parser to HTMl
$ npm start
// view the html file
$ anywhere

```
