nodeps
==============================
A general purpose Photoshop file parser to HTMl file.

## Installation

clone the repository and install the dependencies.

```sh
git@github.com:nodeps/nodeps.git
cd nodeps
npm install 
```

## Execution

Copy a psd file to the root directory.
Example: h5.psd

```sh
// PSD parser to HTMl
$ npm start
// view the html file
$ anywhere

```
